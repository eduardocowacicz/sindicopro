# Projeto Revisado

Esta pasta concentra a documentacao principal do sistema de gestao condominial. Ela funciona como a base oficial para entender o escopo do MVP, as regras do negocio e a orientacao usada na criacao dos mockups.

## Arquivos desta pasta

| Arquivo | Finalidade |
| --- | --- |
| [ResumoDeFuncoesRevisado.txt](ResumoDeFuncoesRevisado.txt) | Visao executiva do projeto, com objetivo, prioridades do MVP e funcoes principais |
| [LevantamentoDeRequisitosRevisado.txt](LevantamentoDeRequisitosRevisado.txt) | Requisitos funcionais, regras de negocio, requisitos nao funcionais e itens fora do escopo |
| [PromptPlanejamentoDesenvolvimento.txt](PromptPlanejamentoDesenvolvimento.txt) | Direcionamento usado para planejar a experiencia e as telas dos mockups |

## Ordem recomendada de leitura

1. Comece pelo [ResumoDeFuncoesRevisado.txt](ResumoDeFuncoesRevisado.txt)
2. Depois leia o [LevantamentoDeRequisitosRevisado.txt](LevantamentoDeRequisitosRevisado.txt)
3. Por fim, consulte o [PromptPlanejamentoDesenvolvimento.txt](PromptPlanejamentoDesenvolvimento.txt) para entender a logica de UX e navegacao

## O que esta consolidado aqui

- objetivo do sistema
- contexto do condominio
- prioridade do MVP
- modulos principais
- regras de acesso e historico
- regras financeiras e de fechamento
- diretrizes de usabilidade, seguranca e auditoria
